# Large Dataset Analysis using Dask

## Objective
Perform analysis on a large dataset using Dask to demonstrate scalable data processing.

## Tools & Technologies
- Python
- Dask
- Pandas
- Google Colab

## Dataset
Sample Superstore Dataset

## Analysis Performed
1. Dataset Overview
2. Sales by Category
3. Profit by Region
4. Top 10 Products by Sales

## Conclusion
Dask efficiently processed the dataset and performed aggregations on large-scale data, demonstrating scalable analytics.
